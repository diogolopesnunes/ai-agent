# ai-agent
An intelligent corporate support assistant integrating Python, traditional machine learning, the OpenAI API, LangChain, LangGraph, RAG, context management, MCP, security, evaluation, and a Streamlit interface. The goal is not merely to produce a visual demonstration, but to deliver a compact, testable, traceable, and secure system.
Um Assistente Inteligente de Suporte Corporativo integrando Python, Machine Learning tradicional, OpenAI API, LangChain, LangGraph, RAG, gestão de contexto, MCP, segurança, avaliação e uma interface em Streamlit. O objetivo não é apenas produzir uma demonstração visual, mas entregar um sistema pequeno, testável, rastreável e seguro.
